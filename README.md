# Excel
Excel projects for data cleaning, analysis, and reporting using formulas, pivot tables, and charts.

This repository contains Excel workbooks I’ve built as a data analyst to explore and present data. The projects focus on cleaning messy datasets, using formulas like XLOOKUP and IF, summarizing data with pivot tables, and creating charts that make insights easy to understand. The goal is to show how Excel can turn raw spreadsheets into clear, decision-ready reports.
